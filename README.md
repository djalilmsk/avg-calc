## CookedCalc

CookedCalc a semester average calculator.

## Features

- Module-based average calculation
- Per-module Exam/TD weights
- Enable/disable Exam or TD in calculation
- Undo, redo, and reset actions
- Snapshot history with rename/restore/delete
- Mobile and desktop layouts

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui components

## Run Locally

```bash
git clone https://github.com/djalilmsk/avg-calc
cd avg-calc

npm install
npm run dev
```
